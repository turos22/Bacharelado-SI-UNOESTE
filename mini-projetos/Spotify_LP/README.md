# 🎵 Spotify Landing Page

Um projeto desenvolvido para praticar e consolidar conhecimentos fundamentais de desenvolvimento web de forma visual e estruturada. O objetivo foi recriar uma landing page responsiva inspirada na interface do Spotify.

---

## 🚀 Tecnologias Utilizadas

O projeto foi construído utilizando tecnologias essenciais para o Front-End, focando na semântica e na estilização fluida:

* **HTML5:** Estruturação semântica de toda a página.
* **CSS3:** Estilizações personalizadas, cores, tipografia e ajustes finos de layout.
* **Bootstrap:** Utilização do sistema de grid, alinhamentos, classes utilitárias e componentes para acelerar o desenvolvimento estrutural e garantir responsividade.

---

## 🎯 Objetivos de Aprendizado

Este repositório foi criado como um ambiente de treino, onde pude focar em:
* Manipular o sistema de colunas e containers do Bootstrap.
* Customizar estilos padrões de frameworks utilizando CSS próprio.
* Garantir uma interface que se adapta bem a diferentes tamanhos de tela (Responsividade).
* Manter a organização de arquivos e diretórios (`css/` e `imagens/`) de forma profissional.
